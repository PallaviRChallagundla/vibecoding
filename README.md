# APP - 1 🌸 Bloom — Spring Home Reset

> A room-by-room spring cleaning ritual for busy women in NJ and NY.

[Live App](https://effervescent-cactus-4b827c.netlify.app)

---

## What it does

Bloom guides you through resetting your home for spring — one room at a time. 
Each room has a candle pairing, flower suggestion, seasonal swap and a full 
task + shopping checklist. There is a 7-day spring challenge, a focus timer with weekend 
warrior mode, and every user's progress saves to the cloud.

---

## Built with

- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Backend:** Supabase (PostgreSQL database, row-level security, JWT authentication)
- **Hosting:** Netlify (CDN, continuous deployment)
- **PWA:** Service worker, web app manifest — installs on iPhone and Android

---

## Features — V1

- 7-day spring challenge with daily task unlocks
- Tasks categorized by efforts and rooms
- Shopping checklists with candle and flower pairings, and items linked
- User authentication with email sign-up and confirmation
- Cloud database sync — progress saves across all dev
